# 動作確認用アプリ

## Usage

``` bash
# Install dependencies
npm install

# 実機で使用してみる場合
tns preview --bundle

# シミュレータで実行する場合
tns run ios --bundle
tns run android --bundle

# Build, watch for changes and debug the application 謎(要検証)
tns debug <platform>

# Build for production 謎(要検証)
tns build <platform> --env.production

```
